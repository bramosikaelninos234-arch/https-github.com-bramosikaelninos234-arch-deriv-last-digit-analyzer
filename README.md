# deriv-last-digit-analyzer

A web app that tracks and analyzes the last digit of Deriv synthetic indices for Over/Under trading strategies.

### **What it does**
- Connects to Deriv API and streams live tick data
- Shows live percentage distribution of digits 0-9 
- Highlights Over/Under stats for any barrier you set
- Displays digit history + streaks to spot patterns

### **Tech Stack**
- Frontend: HTML, CSS, JavaScript / React
- API: Deriv WebSocket API
- Charts: Chart.js
